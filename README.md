# About this Dashboard
This dashboard shows the data of a fitness facility with multiple locations across Michigan. How many members are in each facility and individual data about each of them.

# What this dashboard provides
The business owner can find metrics about: 
- Current members attendance
- Group and individual classes taken
- Drop-ins by external people
- New members
- Lenght of memberships
- Workouts implemented
- Other services like nutrition, private training, training gear sold, suplements and beverages sold
- Revenue by facility - weekly, monthly and annual. 

# Interactions
The business owner can see information about each graphic presented, download and printing individual reports, toggle between training facilities, members and workouts.
